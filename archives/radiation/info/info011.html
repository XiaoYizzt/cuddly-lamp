    当我接受医用 X 光检查时，我会接触到多少辐射？

    X 射线程序中辐射暴露的确切量因接收 X 射线的身体部位而异。
    常见的 X 射线检查程序的一些示例是：

    单次胸部 X 光检查：0.02 mSv (2 mrem)

    牙科 X 光（四个咬合口）：0.004 mSv (0.4 mrem)

    四肢和关节：0.06 mSv (6 mrem)

    腹部：0.7 mSv (70 mrem)

    乳房 X 线照片（四张图像）：0.13 mSv (13 mrem)

    资料来源：美利坚合众国全国辐射防护与测量委员会 (NCRP)，第 160 号报告


    通常，与其他辐射源（例如家中的氡气）相比，X 射线期间接收到的辐射较小。来自自然本底源（用于比较）的年平均辐射剂量为 3.0 mSv (300 mrem)。


    什么是 X 射线以及它们如何用于医学？

    X 射线是以光速在空气中传播的不可见能量波。X 射线能量足够高，一些辐射可以穿过物体，例如内脏、
    身体组织和衣服。在 X 射线程序中，X 射线辐射穿过身体并到达检测器，例如胶片或连接到计算机的电
    子检测器。一般来说，密度较大的物体（如骨骼和钙沉积物）会吸收更多的辐射，从而减少通过探测器
    的辐射量。因此，与密度较小的物体（看起来较暗）相比，密度较大的物体会在检测器上留下图像（也
    就是说，它们看起来更亮）。这就是骨骼在 X 射线图像上呈现白色的原因。放射科医生和其他受过专门
    训练或经验丰富的医生可以阅读这些图像来诊断医疗状况或伤害。


    医用 X 光检查对我来说安全吗？

    通常，与其他辐射源相比，医用 X 射线期间接收到的辐射较小。
    X 光可以提供有关医疗状况的信息，因此与您的医生讨论 X 光
    的益处和风险非常重要。

    如果您一年内已经接受了很多次X光检查，我推荐您把情况告诉医生

    重要的是要知道，现有的最佳科学表明，即使是低水平的辐射暴露也可能会略微增加患癌症的风险。
    因此，与您的医疗专业人员讨论 X 光检查的必要性非常重要，尤其是对于儿童和孕妇。


    我怎样才能最大限度地减少 X 射线辐射暴露？

  • 询问您的医疗保健提供者 X 光检查有何​​帮助。询问是否有其他可能风险较低但仍可进行良好评估或治疗的程序。
  • 如果您怀孕或可能怀孕，请提前告知 X 光工作人员。
  • 询问是否可以使用防护罩。如果您或您的孩子正在接受 X 光检查，请询问是否应使用铅围裙或其他防护罩。
  • 了解您的 X 光检查史。保留一份影像记录清单，包括牙科 X 光片。向您的医疗保健专业人员出示该卡，以避免对同一身体部位进行不必要的重复 X 光检查。为您家中的每个人保留一张记录卡。
